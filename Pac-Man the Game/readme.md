Application written in HTML5

## 1. Modularity and Readability
- Game logic is separated into the `pacman.js` file.
- HTML handles only structure and styling, making maintenance easier.

## 2. Use of HTML5 Canvas
- The game runs without additional plugins.
- Canvas enables smooth animations and real-time interactions.

## 3. Ergonomics and Aesthetics
- Minimalist design keeps the focus on the game.
- Center alignment and black background create a retro atmosphere.

## 4. Scalability and Portability
- Code works in all modern browsers.
- Easy to embed the game into other web applications.

## 5. Extensibility
- Custom font and audio increase immersion.
- Easy to add menus, score counters, or extra styles.

# ✅ Plusy kodu Pacman (HTML5)

## 1. Modularność i czytelność
- Logika gry znajduje się w osobnym pliku `pacman.js`.
- HTML odpowiada tylko za strukturę i styl, co ułatwia utrzymanie.

## 2. Wykorzystanie HTML5 Canvas
- Gra działa bez dodatkowych pluginów.
- Canvas zapewnia płynne animacje i interakcje w czasie rzeczywistym.

## 3. Ergonomia i estetyka
- Minimalistyczny design skupia uwagę na grze.
- Centralne wyrównanie i czarne tło nadają klimat retro.

## 4. Skalowalność i przenośność
- Kod działa w każdej nowoczesnej przeglądarce.
- Łatwo osadzić grę w innych aplikacjach webowych.

## 5. Rozszerzalność
- Dodanie czcionki i audio zwiększa immersję.
- Możliwość łatwego dodania menu, liczników punktów czy dodatkowych stylów.
